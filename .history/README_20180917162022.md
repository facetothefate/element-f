
## Basic usage
```javascript

    let testComp = component((data, states) => {
        return element("div",{},
            element("h1", {text:data.header}),
            element("p", {
                text: data.body,
                style: {
                    lineHeight: "15px"
                }
            }),
            stateElement(
                (states) => {
                    if (states.haveSub) {
                        return element("p", {
                            text: data.sub
                        });
                    }
                }
            ),
        );
    });

    testComp({
        header: "Hello world!",
        body: "",
        haveSub: false
    },{});
```
